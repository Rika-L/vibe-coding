# 睡眠记录增删改查功能

## 需求概述

支持用户手动添加、编辑、删除睡眠记录，在历史记录页面提供完整的 CRUD 操作。

## 功能设计

### 首页入口

- 位置：上传区域下方
- 样式：小号链接文字 "或 手动添加记录"
- 点击后弹出添加记录对话框

### 添加/编辑记录对话框

**字段列表：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 日期 | 日期选择器 | 是 | 睡眠日期 |
| 睡眠时长 | 数字输入 | 是 | 单位：小时 |
| 入睡时间 | 时间选择器 | 否 | 格式：HH:mm |
| 起床时间 | 时间选择器 | 否 | 格式：HH:mm |
| 深睡时长 | 数字输入 | 否 | 单位：小时 |
| 浅睡时长 | 数字输入 | 否 | 单位：小时 |
| REM 时长 | 数字输入 | 否 | 单位：小时 |
| 清醒次数 | 数字输入 | 否 | 单位：次 |
| 睡眠评分 | 数字输入 | 否 | 范围：0-100 |
| 心率 | 数字输入 | 否 | 单位：bpm |

**交互：**
- 添加模式：标题为"添加睡眠记录"，保存后创建新记录
- 编辑模式：标题为"编辑睡眠记录"，保存后更新现有记录

### 历史记录页面改造

**表格新增操作列：**
- 每行末尾添加操作按钮
- 编辑按钮（铅笔图标）：点击打开编辑对话框
- 删除按钮（垃圾桶图标）：点击弹出删除确认对话框

**删除确认：**
- 使用 AlertDialog 组件
- 提示"确定要删除这条睡眠记录吗？"
- 确认后删除记录并刷新列表

## 技术实现

### API 端点

**创建记录：**
```
POST /api/sleep-records
Body: { date, sleepDuration, bedTime?, wakeTime?, deepSleep?, lightSleep?, remSleep?, awakeCount?, sleepScore?, heartRate? }
Response: { record }
```

**获取单条记录：**
```
GET /api/sleep-records/[id]
Response: { record }
```

**更新记录：**
```
PUT /api/sleep-records/[id]
Body: { date?, sleepDuration?, ... }
Response: { record }
```

**删除记录：**
```
DELETE /api/sleep-records/[id]
Response: { success: true }
```

## 边界情况

1. **日期重复**：同一日期只能有一条记录，添加时检查是否已存在
2. **数据验证**：睡眠时长 > 0，评分范围 0-100，心率范围 30-200
3. **权限检查**：只能操作自己的记录
4. **删除保护**：删除前确认，防止误操作

# 睡眠记录批量删除/清空功能设计

**日期:** 2026-04-02

## 功能概述

在历史页面的"睡眠记录"标签页添加批量删除和清空全部功能。

## 涉及改动

### 1. 前端页面 (`src/app/history/page.tsx`)

- 表格新增复选框列（表头 + 每行）
- 顶部操作栏：显示已选 X 条记录 + 批量删除按钮 + 清空全部按钮
- 批量删除确认对话框：显示即将删除的记录数量
- 清空全部确认对话框：红色警示 + 需输入"确认清空"文字

### 2. 后端 API (`src/app/api/sleep-records/route.ts`)

- `DELETE /api/sleep-records` - 批量删除（接收 `{ ids: string[] }`）
- `DELETE /api/sleep-records?all=true` - 清空当前用户所有记录

## 交互流程

1. 用户勾选记录 → 顶部显示"已选 X 条"
2. 点击"批量删除" → 确认对话框 → 确认后删除并刷新
3. 点击"清空全部" → 红色警示确认框 + 输入"确认清空" → 确认后删除并刷新

## 确认对话框设计

### 批量删除确认
- 标题: "确认删除选中的记录？"
- 内容: "即将删除 X 条睡眠记录，此操作无法撤销。"

### 清空全部确认
- 标题: "确认清空所有记录？"
- 内容: "此操作将删除所有睡眠记录，且无法恢复！"
- 输入框: 需输入"确认清空"才能点击确认按钮

## 验收标准

1. ✅ 可以勾选单条/多条记录
2. ✅ 批量删除选中的记录
3. ✅ 一键清空所有记录
4. ✅ 删除后数据正确刷新
5. ✅ 清空全部需要二次确认（输入文字）

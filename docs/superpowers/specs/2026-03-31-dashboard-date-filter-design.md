# Dashboard 日期区间筛选功能

## 需求概述

在 Dashboard 页面添加日期区间筛选器，用户可以选择日期范围查看数据，避免数据太多时显示混乱。

## 功能设计

### 默认行为
- 默认显示全部数据（保持当前行为）
- 用户可通过筛选器选择日期范围

### UI 设计
- 位置：统计卡片上方
- 组件：两个日期选择器 + 筛选按钮 + 清除按钮
- 样式：参考历史记录页面的筛选器

### 交互流程
1. 用户选择开始日期和结束日期
2. 点击"筛选"按钮
3. 数据刷新为选定范围内的记录
4. 统计数据和图表更新
5. 点击"清除"恢复显示全部数据

## 技术实现

### 前端修改
**文件**: `src/app/dashboard/page.tsx`
- 添加 startDate、endDate 状态
- 添加日期选择器 UI
- 修改 fetchData 函数传递日期参数

### 后端修改
**文件**: `src/app/api/sleep-data/route.ts`
- 接收 startDate、endDate 查询参数
- 根据日期范围过滤数据

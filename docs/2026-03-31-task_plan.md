## ⚠️ 注意事项

1. **数据库迁移**: 已完成 `npx prisma migrate dev --name add_user_auth`
2. **JWT Secret**: 已在 `.env` 中配置 `JWT_SECRET`
3. **现有数据**: 现有 SleepRecord 数据的 userId 为 null（允许未关联用户的历史数据）
4. **Cookie 安全**: JWT 存储在 HttpOnly Cookie 中
5. **密码安全**: 使用 bcryptjs 进行密码哈希

## 🔐 安全考虑

- 密码使用 bcrypt 哈希存储
- JWT 存储在 HttpOnly Cookie 防止 XSS
- 使用 Zod 进行输入验证防止注入攻击
- 登录状态通过中间件保护敏感路由
